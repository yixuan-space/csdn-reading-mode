# CSDN 沉浸阅读助手

一个为 CSDN 博客文章页面提供 Notion 风格阅读体验和 PDF 导出功能的 Tampermonkey 脚本。

## 功能

- **沉浸式阅读模式**：一键移除 CSDN 页面中的所有广告、侧边栏等干扰元素，提供类似 Notion 的清爽阅读体验
- **PDF 导出功能**：将文章导出为 PDF 格式，保持良好的排版和可读性
- **图片居中显示**：自动将图片居中对齐，提升阅读体验

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 [安装脚本](./csdn-reading-mode.user.js)
3. 访问任意 CSDN 博客文章页面即可使用

## 使用

- **阅读模式**：点击页面右上角的「阅读模式」按钮进入/退出阅读模式
- **导出 PDF**：点击「导出 PDF」按钮将当前文章保存为 PDF 文件
